# Оценка кредитоспособности заёмщиков

Проект нацелен на решение финансовых вопросов в сфере кредитования: учитывая возраст, уровень доходов, семейное положение и т.д. установить, выдавать ли кредит потенциальному заёмщику, и если да, то какую сумму, под какую процентную ставку и на какой срок.

Для решения данной задачи используется модель глубокого обучения.
